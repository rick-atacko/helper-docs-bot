# Helper Docs Bot

Telegram-бот для поиска документов, работы с базой знаний и управления дедлайнами.

## Цель проекта

Снизить время поиска типовых документов и регламентов внутри проектных команд.

## Текущий функционал

* Приветствие пользователя
* Telegram-интерфейс
* Базовая архитектура проекта

## Планируемый функционал

* Поиск документов по ключевым словам
* База знаний с ответами на вопросы
* Управление дедлайнами
* Напоминания пользователям
* Интеграция с OpenAI API

## Технологии

* Python
* Aiogram
* SQLite
* OpenAI API
* Git
* GitHub
