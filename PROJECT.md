Бот-ассистент для поиска необходимых документов и информации проектным командам
Цель: упростить хранение и выдачу документации, уменьшить потребность в личных письмах о необходимых образцах документов

Функции MVP:
- приветствие пользователя
- поиск документов
- база знаний
- создание дедлайнов
- напоминания

Стек:
- Python
- Aiogram
- SQLite
- OpenAI API
